DictionaryEntryExtension

DictionaryEntryExtension extension Null safety

A DictionaryEntry is a unit of entry in the Dictionary of an inverted index. It enumerates the following property getters:

  • term is the word/term that is indexed; and
  • dFt is the number of documents that contain term.
