InMemoryIndexer

InMemoryIndexer class Null safety

Maintains an in-memory inverted index:

  • dictionary is the in-memory term dictionary for the indexer;
  • postings is the in-memory postings hashmap for the indexer; and
  • tokenizer is the Tokenizer instance used by the indexer to parse documents to tokens. Defaults to Indexer.kDefaultTokenizer.

Provides the index method to allow documents to be indexed one-by-one, adding the terms to the dictionary and the postings for each document to the postings.
