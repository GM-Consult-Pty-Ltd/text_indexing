DocumentPostings

DocumentPostings class Null safety

The DocumentPostings class enumerates the properties of a document posting in a PostingsMap as part of an inverted index of a dataset:

  • term is the word/term that is indexed;
  • docId is the document's id value;
  • positions is the zero-based list of word positions of the term in the document;
