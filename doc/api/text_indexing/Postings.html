Postings

Postings typedef Null safety

Postings = Map<String, Map<String, List<int>>>

Defines a collection of postings for a positional inverted index.

Postings is a hashmap of PostingsEntrys with the vocabulary as key and the postings lists for the linked documents as values.

A Postings can be an asynchronous data source or an in-memory hashmap.
