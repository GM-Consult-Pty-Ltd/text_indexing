InMemoryIndex

InMemoryIndex class Null safety

The InMemoryIndex is an implementation of the InvertedIndex interface that mixes in InvertedIndexMixin and InMemoryIndexMixin.

The InMemoryIndex is intended for fast indexing of a smaller corpus using in-memory dictionary, k-gram and postings hashmaps.
