Returns a list of DictionaryEntrys from the entries in the Dictionary.

Sorts the terms according to sortBy value:

  • TermSortStrategy.byTerm sorts the DictionaryEntrys alphabetically (default); or
  • TermSortStrategy.byFrequency sorts the DictionaryEntrys by Ft in descending order.
