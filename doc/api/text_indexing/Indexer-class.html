Indexer

Indexer class Null safety

Base class for classes that construct and maintain an inverted index:

  • tokenizer is the Tokenizer instance used by the indexer to parse documents to tokens;
  • the index method indexes documents one-by-one, adding a list of TermPositions to the postingsStream for each document;
  • listen to postingsStream to update your term dictionary and postings map; and/or
  • override emit to perform additional actions whenever a document is indexed, e.g. updating a term dictionary of postings list.
