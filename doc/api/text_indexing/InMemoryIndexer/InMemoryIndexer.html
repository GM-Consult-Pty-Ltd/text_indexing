Initializes a InMemoryIndexer instance:

  • pass a tokenizer to parse text to tokens, or use the default TextIndexer.kDefaultTokenizer;
  • pass a jsonTokenizer used to parse JSON documents to tokens, or use the default TextIndexer.kDefaultJsonTokenizer;
  • pass an in-memory dictionary instance, otherwise an empty Dictionary will be initialized.
  • pass an in-memory postings instance, otherwise an empty Postings will be initialized.
