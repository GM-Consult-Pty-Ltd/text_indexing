Returns a list of DftMapEntrys from the entries in the DftMap.

Sorts the terms according to sortBy value:

  • TermSortStrategy.byTerm sorts the DftMapEntrys alphabetically (default); or
  • TermSortStrategy.byFrequency sorts the DftMapEntrys by Ft in descending order.

Implementation

List<MapEntry<String, double>> toList(
    [TermSortStrategy sortBy = TermSortStrategy.byTerm]) {
  final list = entries.toList();
  switch (sortBy) {
    case TermSortStrategy.byTerm:
      list.sort(((a, b) => a.key.compareTo(b.key)));
      break;
    case TermSortStrategy.byFrequency:
      list.sort(((a, b) => b.value.compareTo(a.value)));
      break;
  }
  return list;
}
