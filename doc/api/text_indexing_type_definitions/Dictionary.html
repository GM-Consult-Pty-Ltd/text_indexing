Dictionary

Dictionary typedef Null safety

Dictionary = Map<Term, Ft>

Defines a term dictionary used in an inverted index.

The Dictionary is a hashmap of DictionaryEntrys with the vocabulary as key and the document frequency as the values.

A Dictionary can be an asynchronous data source or an in-memory hashmap.

Alias for Map<String, int>.
